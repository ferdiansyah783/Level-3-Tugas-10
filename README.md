## Getting Started

```bash
npm i

npx prisma init

npx prisma migrate dev --name init

npx prisma generate
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
